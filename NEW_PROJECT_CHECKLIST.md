## Troubleshooting

### Issue: "Project creation failed"
- Try a different project name
- Check your Supabase account limits
- Contact Supabase support

### Issue: "Migration failed"
- Verify database is clean (no existing tables)
- Check database password is correct
- Review error message in terminal

### Issue: "Can't link CLI to project"
- Verify project reference is correct
- Ensure you have owner/admin access
- Try: `supabase link --project-ref REF --password PASSWORD`

### Issue: "Admin user can't login"
- Verify UUID matches between auth.users and admissions_officers
- Check "Auto Confirm User" was enabled
- Verify email is correct
